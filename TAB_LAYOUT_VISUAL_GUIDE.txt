<!-- Key Features Illustrated -->

✓ ALL TABS VISIBLE - No horizontal scrolling needed
✓ FLEX DISTRIBUTION - Tabs grow/shrink to fit viewport
✓ RESPONSIVE PADDING - Adjusts based on screen size
✓ TEXT TRUNCATION - Long names show ellipsis (...)
✓ NO WRAPPING - Always horizontal, never stacks

<!-- CSS Behavior -->

Screen Width: 1200px
├─ Tab 1: flex-grow fills available space
